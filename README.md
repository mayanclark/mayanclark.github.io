# mayanclark.github.io

Portfolio site for Maya Clark — Environmental GIS Analyst, Grant Writer, and Land Advocate.

Live at: [mayanclark.github.io](https://mayanclark.github.io)

## To update content

All content is in `index.html`. Search for the section you want to edit:

- **Hero stats** — search `stat-card`
- **About / bio text** — search `about-body`
- **Awards & credentials** — search `award-card`
- **Projects** — search `project-card`
- **Services** — search `service-card`
- **Contact info** — search `contact-link`

## To add your email

Replace `hello@mayanclark.com` with your real email address.

## Built with

Plain HTML + CSS — no build tools, no frameworks, no dependencies except Google Fonts.
